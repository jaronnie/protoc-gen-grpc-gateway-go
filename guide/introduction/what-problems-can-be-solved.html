# what-problems-can-be-solved

  • 一键自动生成 httpsdk 客户端代码, 无需维护
  • 支持将多个微服务的客户端 httpsdk 整合
  • 根据路由能对资源进行分类, 让客户端调用更加方便
  • 能对资源接口进行扩展, 如支持 websocket 接口以及支持扩展 http 原生框架的接口
  • 能自动生成 mock 代码, 提升单测覆盖率
