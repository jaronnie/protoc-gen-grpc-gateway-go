# what-problems-can-be-solved

  • 一键自动生成 go 客户端代码, 无需人工维护
  • 支持将多个微服务的客户端整合
  • 根据 service 或者根据路由进行资源分类
  • 能对资源接口进行扩展, 如支持 websocket 接口以及支持扩展 http 原生框架的接口
  • 能自动生成 mock 代码, 让单元测试更加简单
